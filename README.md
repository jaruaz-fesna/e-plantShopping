# e-plantShopping — Paradise Nursery

A React + Redux e-commerce web application for browsing and purchasing houseplants online.

## Project Name

**Paradise Nursery** — *Where Green Meets Serenity*

## Description

Paradise Nursery is a fully functional plant shopping app built with React and Redux Toolkit. Users can browse plants grouped by category, add them to a shopping cart, manage quantities, and proceed to checkout.

## Features

- Landing page with company name, tagline, and background image
- Product listing page with 5 plant categories (30+ plants total)
- Add to cart with button disable after selection
- Shopping cart page with quantity controls, item deletion, and total cost calculation
- Dynamic cart icon showing live item count
- Navbar with navigation to Home, Plants, and Cart
- Checkout placeholder ("Coming Soon")
- Deployed via GitHub Pages

## Tech Stack

- React 18
- Redux Toolkit
- Vite
- CSS

## Getting Started

```bash
cd e-plantShopping
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The app is deployed using GitHub Pages.
